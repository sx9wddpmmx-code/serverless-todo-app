# Serverless Todo Web Application

Dieses Repository enthält eine einfache ToDo-Webanwendung, die zur Demonstration einer serverlosen Webarchitektur entwickelt wurde. Die Anwendung dient als praktisches Beispiel im Rahmen des Moduls *Master Konferenzseminare* und begleitet ein wissenschaftliches Paper zum Thema serverlose Webarchitekturen.

Ziel der Anwendung ist es, zentrale Konzepte der serverlosen Webentwicklung anhand einer überschaubaren ToDo-Anwendung zu veranschaulichen. Im Fokus stehen dabei die Trennung von Frontend und Backend, die Nutzung serverloser Funktionen sowie deren Integration in eine moderne Webanwendungsarchitektur. Auf komplexe Geschäftslogik wurde bewusst verzichtet, um die architektonischen Aspekte klar hervorzuheben.

---

## Projektübersicht

Die Anwendung besteht aus zwei zentralen Teilen:

- einem Frontend als Single-Page-Application
- einem serverlosen Backend in Form von HTTP-basierten Funktionen

Frontend und Backend werden gemeinsam über die Plattform **Vercel** betrieben und automatisch deployt.

---

## Deployment

- Automatisches Deployment über Vercel
- Jeder Commit im Repository triggert einen neuen Build
- Frontend und Backend werden gemeinsam deployed

---

## Hinweis

Dieses Projekt dient ausschließlich Demonstrations- und Analysezwecken. Aspekte wie Authentifizierung, Rollenmanagement oder Zugriffsbeschränkungen wurden bewusst nicht umgesetzt, da sie nicht im Fokus der Ausarbeitung stehen.
